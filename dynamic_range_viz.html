About Dynamic Range in Wireless Systems

This visualization demonstrates the dynamic range of signal propagation in a multi-RU (Radio Unit) wireless uplink system. The dynamic range refers to the variation in received signal strength across different distances from the transmitter, which is a critical consideration in cell-free and distributed antenna systems.

Large-Scale Fading Coefficient (LSFC): The path loss model uses LSFC = 1/(1+(d/d₀)^α), where d is the distance, d₀ is the reference distance, and α is the path loss exponent. This model captures how signal power decreases with distance due to propagation effects.

Tiered Architecture: The system divides coverage into three tiers based on distance from the transmitter:

  • 1st Tier (Green): Closest RUs with highest SNR - these provide the strongest signal reception
  • 2nd Tier (Yellow): Middle-distance RUs with moderate SNR - secondary coverage layer
  • 3rd Tier (Red): Farthest RUs with lowest SNR - edge coverage extending to 0 dB threshold

Key Parameters: The noise floor is set at N₀ = -169 dBm/Hz (thermal noise), and the SNR is calculated as SNR = (P_tx × LSFC) / (N₀ × BW), where P_tx is transmit power and BW is bandwidth. Adjust the sliders to explore how tier distances and target SNR levels affect the required system parameters like path loss exponent and reference distance.

The contour map shows the spatial distribution of SNR around a centrally-located transmitter, revealing how signal quality varies across a 2D plane. The SNR vs Distance plot illustrates the signal degradation as a function of range, with tier boundaries marked to show the three-tier coverage architecture.
