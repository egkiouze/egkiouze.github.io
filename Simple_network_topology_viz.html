📖 About the Network Topology:

This visualization implements a triangular-hexagonal network pattern based on your MATLAB code. Radio Units (RUs) are placed in a hexagonal grid pattern for optimal coverage. Location Centers form triangular regions between RUs, with each location served by exactly 3 surrounding RUs. User Equipment (UEs) are randomly distributed within their location triangles.

The hexagonal pattern provides uniform coverage with minimal overlap, while triangular locations ensure that each user is always within range of multiple RUs for diversity and handover. This topology is commonly used in cellular networks for efficient spectrum usage and load balancing.

🖱️ Interaction: Drag to pan, use the zoom slider, or scroll to zoom. Click regenerate to randomly redistribute users within their locations.
